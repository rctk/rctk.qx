To start working/developing with rctk.qooxdoo you will need to:

1. download qooxdoo into rctk.qx. This can be done using the install
   script in the qooxdoo directory:

   $ cd qooxdoo
   $ ./install.sh

   This will download and unpack qooxdoo

2. build: 
   $ qooxdoo/qooxdoo-1.3-sdk/tool/bin/generator.py
   ...
